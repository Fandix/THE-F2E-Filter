# 使用React.js實現一個小型的觀光網站
使用axios來達到非同步請求網路數據的功能並利用Redux達到統一管理State，將React的Component劃分為負責處理數據的Container Components和負責選染UI的
Presentational Components，將兩種組件分開管理以方便之後的維護與更改。
